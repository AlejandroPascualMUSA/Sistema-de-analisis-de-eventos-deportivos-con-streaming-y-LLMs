# Mallorca

Mallorca se interpreta desde estructura defensiva, duelos, juego directo y ataques de menor volumen pero potencialmente concretos. En StatsBomb conviene revisar despejes, faltas, recuperaciones, pases largos, tiros y xG.

Si Mallorca tiene pocos eventos ofensivos pero xG significativo, el informe puede destacar eficiencia o jugadas aisladas de valor. Si el pico de intensidad contiene despejes, puede ser una fase defensiva.
