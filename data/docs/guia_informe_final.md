# Guía del informe final

El informe final combina evidencia cuantitativa y contexto documental. La estructura recomendada es resumen del stream, métricas por equipo, jugadores destacados, tramos de intensidad, alineaciones, contexto RAG y conclusión técnica.

La separación de fuentes debe quedar clara: Spark calcula cifras desde eventos, RAG aporta contexto textual y LangGraph coordina las herramientas para producir Markdown y HTML.
