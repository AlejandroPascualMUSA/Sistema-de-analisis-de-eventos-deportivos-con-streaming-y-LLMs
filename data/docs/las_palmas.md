# Las Palmas

Las Palmas se interpreta como un equipo que suele buscar salida elaborada, posesión y progresión con balón. En StatsBomb conviene mirar volumen de pases, acierto, pérdidas, entradas al último tercio y xG.

Este documento solo debe aparecer cuando Las Palmas participa. Si aparece en otro partido, el filtro RAG debe descartarlo.
