# Valencia

Valencia se interpreta como un equipo que puede combinar energía competitiva, transiciones, progresión exterior y fases de repliegue. En StatsBomb conviene mirar recuperaciones, conducciones, pases verticales, faltas, tiros, xG y tramos de intensidad para entender cuándo acelera el partido.

Si Valencia genera menos posesión pero xG similar o superior, el informe puede destacar eficiencia en ataques, transiciones o acciones de mayor valor. Si sus picos de intensidad contienen despejes y faltas, el contexto probablemente sea defensivo; si contienen tiros y carries, puede ser ofensivo.
