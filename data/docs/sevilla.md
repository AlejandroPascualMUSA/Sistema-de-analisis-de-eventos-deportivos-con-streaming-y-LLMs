# Sevilla

Sevilla se interpreta como un equipo con tradición de intensidad, uso de bandas, centros y fases competitivas de ida y vuelta. En StatsBomb conviene revisar centros, tiros, xG, faltas, recuperaciones y tramos de actividad.

Este documento solo debe usarse cuando Sevilla participa en el partido. Si no participa, el filtro RAG debe descartarlo para evitar contaminar el informe con contexto de otro equipo.
