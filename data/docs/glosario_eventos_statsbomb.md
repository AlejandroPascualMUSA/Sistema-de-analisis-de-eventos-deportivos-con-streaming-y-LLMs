# Glosario de eventos StatsBomb

Pass representa un pase; Shot representa un tiro; Pressure representa una presión; Ball Recovery indica recuperación; Carry representa conducción; Dribble registra regate; Interception indica interceptación; Foul Committed y Foul Won separan faltas cometidas y recibidas.

Cada tipo de evento aporta información diferente. Para el informe, los eventos ofensivos, defensivos y de intensidad se agrupan para resumir participación, amenaza y fases de partido.
